### API: Julie Blog

* * *
  
To see result,
1. download repo
2. `npm install`
3. `npm run develop`
4. make API calls to `localhost:1337` + endpoints

* * *
**ENDPOINTS**

- `/articles` for all articles
- `/articles/id` for a specific article with id
- `/articles?slug="slug"` for a specific article with slug
